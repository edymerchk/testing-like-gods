# Testing like the gods

## rspec-instafail

### Installation

> Add rspec-instafail` to :test group in the Gemfile:

```ruby
group  :test do
  gem 'rspec-instafail'
end
```

> Download and install by running:

```
bundle install
```

Add to `.rspec` file:

--require rspec/instafail

--format RSpec::Instafail

> Use rspec command as usual
